## Висновки

Емпіричне тестування підтвердило теоретичні оцінки складності алгоритмів.

Insertion Sort демонструє квадратичне зростання часу і є непридатним для великих масивів.

Merge Sort стабільно працює за O(n log n), але поступається Timsort через накладні витрати на рекурсію та памʼять.

Timsort показав найкращі результати у всіх тестах завдяки поєднанню сортування вставками та злиттям, а також оптимізаціям для частково відсортованих даних.

Саме тому вбудовані функції Python (`sorted`, `sort`) є рекомендованими для використання на практиці.
